# Upload Secure Bundle Instructions

## Option 1: GitHub (Recommended)
1. Create a new private repository on GitHub
2. Upload your `secure-connect-socialnetwork.zip` file
3. Get the raw URL: 
   `https://raw.githubusercontent.com/YOUR_USERNAME/YOUR_REPO/main/secure-connect-socialnetwork.zip`

## Option 2: Dropbox
1. Upload the zip file to Dropbox
2. Create a share link
3. Replace `dl=0` with `dl=1` at the end of URL
